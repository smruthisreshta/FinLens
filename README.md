FinLens – Financial Analysis Portal 📊

A modern AI-powered financial analytics dashboard built using HTML, Tailwind CSS, JavaScript, Chart.js, XLSX, and jsPDF.
FinLens helps businesses and individuals upload financial data, analyze revenue and expenses, detect spending leaks, forecast cash flow, benchmark against industries, and generate professional PDF reports.
